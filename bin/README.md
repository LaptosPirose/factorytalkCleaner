# FactoryTalkCleaner

Ferramenta para limpeza, normalização e análise de bancos de alarmes do FactoryTalk Alarm & Events.

## Objetivo

O FactoryTalkCleaner tem como objetivo reduzir ruídos em bases históricas de alarmes, removendo eventos redundantes e gerando dados mais consistentes para análise operacional.

## Funcionalidades

- Remoção de alarmes redundantes
- Normalização de eventos
- Estatísticas de ocorrência
- Exportação de resultados
- Preparação de dados para análise posterior

## Stack Tecnológica

- Java 26
- Spring Boot
- Maven
- JUnit (testes)
- Git/GitHub (controle de versão)

## Como executar

```bash
./mvnw spring-boot:run
```
